# Guia de Utilização - NannyMeal

## Lista de Compras com Supermercados

A app NannyMeal permite exportar a sua lista de compras para os principais supermercados portugueses.

### Supermercados Disponíveis

| Supermercado | Cor | Website |
|-------------|-----|---------|
| Continente | Azul (#0073A7) | continente.pt |
| Pingo Doce | Verde (#00A04A) | pingodoce.pt |
| Auchan | Vermelho (#E30613) | auchan.pt |
| Intermarché | Laranja (#FF6600) | intermarche.pt |
| Super Save | Laranja (#FF6B35) | supersave.pt |

### Como Exportar a Lista de Compras

1. **Gera um plano semanal** na secção "Plano"
2. **Acede à Lista de Compras** através do menu inferior
3. **Clica no botão "store"** (ícone de loja) no topo direito
4. **Escolhe o supermercado** pretendido
5. O website do supermercado abre automaticamente com os ingredientes pesquisados

### Funcionalidades

- **Tradução automática**: Ingredientes em inglês são traduzidos para português
- **Limpeza automática**: Unidades de medida (g, kg, ml, etc.) são removidas
- **Pesquisa direta**: Os ingredientes são pesquisados diretamente no site do supermercado
- **Comparação de preços**: O Super Save permite comparar preços em várias lojas

### Nota sobre Apps Móveis

Os deep links para os apps móveis (auchan://, continente://, etc.) funcionam apenas em dispositivos móveis com os apps instalados. No browser, a app abre automaticamente o website.

## Dúvidas Frequentes

**P: Por que os ingredientes aparecem em inglês?**
R: As receitas são importadas da API Spoonacular. A app tenta traduzir automaticamente para português.

**P: Posso editar a lista de compras?**
R: Currently, a lista é gerada automaticamente a partir do plano semanal. Podes marcar/desmarcar itens.

**P: Os preços são precisos?**
R: Os preços shown são estimativas baseadas nas receitas. Para preços reais, consulta o website do supermercado.
