# NannyMeal PWA

NannyMeal é um Planeador de Refeições Progressivo (PWA) desenhado para famílias, focado em reduzir o stress do jantar, minimizar o desperdício alimentar e otimizar as compras semanais.

## 🚀 Funcionalidades Modernas

- **PWA (Progressive Web App)**: Funciona no telemóvel como uma app nativa.
- **Onboarding Wizard 2.0**: Configuração detalhada da família (adultos, crianças por idade), preferências dietéticas e restrições (alergias/exclusões).
- **Algoritmo de Geração**: Cria planos semanais baseados no teu tempo disponível e nível de cozinha.
- **Dashboard Premium**: Interface limpa e intuitiva com Material Symbols.

## 🛠️ Stack Tecnológica

- **Frontend**: Vanilla JS, Tailwind CSS.
- **Backend/Auth**: Firebase (Authentication, Firestore, Hosting).
- **Icons**: Material Symbols Outlined.

## 📦 Como clonar e correr

1. **Clonar o repo**:

   ```bash
   git clone <teu-url-do-github>
   cd NannyMeal
   ```

2. **Firebase CLI**:
   Instala o Firebase Tools se ainda não tiveres:

   ```bash
   npm install -g firebase-tools
   ```

3. **Login e Inicialização**:

   ```bash
   firebase login
   firebase init
   ```

4. **Deploy**:
   ```bash
   firebase deploy
   ```
