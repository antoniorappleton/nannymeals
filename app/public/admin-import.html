Importar Receitas

Pesquise e importe receitas da API Spoonacular. Use a Query de Busca para especificar o que procura:

  • peixe ou salmon - receitas de peixe
  • carne ou chicken - receitas de carne
  • sopa - sopas
  • bolo ou dessert - sobremesas

Pode combinar filtros: Tipo, Dieta, Intolerâncias.

science

Teste

Executa uma importação de teste com parâmetros predefinidos (pasta vegetariana, 10 receitas) para verificar se tudo funciona.

sync

Migrar

Atualiza dados de planos de refeições existentes. Necessário apenas uma vez para migrar membros de planos antigos para o novo formato.

payments

Conversão de Preços

Os preços da Spoonacular são em centavos USD. A conversão para EUR (com taxa de inflação de 2.3%) é aplicada automaticamente durante a importação.

Fórmula: preço_eur = (preço_usd / 100) * 0.92 * 1.023
